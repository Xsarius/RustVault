# RustVault User Guide

Welcome to the **RustVault** user guide — your reference for setting up, using, and managing your self-hosted personal finance platform.

## What is RustVault?

RustVault is a self-hosted personal finance application built with an **automation-first** philosophy. Instead of manually entering every coffee purchase, you import bank statements in bulk and let RustVault's rules engine categorize them for you.

### Key Features

- **Bulk import** — CSV, MT940, OFX, QIF, CAMT.053, XLSX, JSON, and more
- **Auto-categorization** — Rules engine learns your patterns and categorizes transactions automatically
- **Budgeting** — Envelope-style budgets with forecasting
- **Multi-currency** — Automatic exchange rate conversion
- **Multi-user** — Shared household finances with role-based access
- **Reports & charts** — Interactive financial visualizations
- **AI-optional** — Receipt scanning and smart categorization (toggleable)
- **Mobile-ready** — iOS and Android apps via Capacitor
